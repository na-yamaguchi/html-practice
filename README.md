## Build Setup

Node.jsをインストールした後にこのディレクトリまで移動してきて以下のコマンドを実行します。

``` bash
# install dependencies
$ npm i

# serve with hot reload at localhost:3000
$ npm run dev

# build for production and launch server
$ npm run build
$ npm run start

# generate static project
$ npm run generate
```
